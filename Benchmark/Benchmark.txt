# 🎯 Benchmark — Frameworks d’interfaces Java

## Objectif
Choisir un framework Java adapté à la création de l’interface graphique du projet **Gestion de Billetterie**.

---

## ⚙️ Frameworks comparés

### 🟦 JavaFX
**Type :** Desktop  
**Langage :** Java / FXML  
**Points forts :**
- Intégré à Java (facile à utiliser)
- Rendu moderne, support CSS & Scene Builder
- Léger, rapide pour du local

**Limites :**
- Pas orienté web
- Nécessite installation sur chaque poste

---

### 🟩 Vaadin Flow
**Type :** Web full Java  
**Langage :** Java uniquement  
**Points forts :**
- Interface web responsive, sans JS
- Compatible Spring Boot
- Déploiement unique sur serveur

**Limites :**
- Installation plus lourde
- Moins performant pour usage local

---

## ✅ Recommandation
- **Choix principal : Vaadin Flow** → idéal pour une application web multi-utilisateurs.  
- **Alternative : JavaFX** → adapté à une exécution locale rapide.  

**Conclusion :** Vaadin Flow offre plus de flexibilité et de portée pour une billetterie accessible depuis navigateur.
